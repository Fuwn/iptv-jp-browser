  <h1>Japanese IPTV Browser</h1>

Browse the latest Japanese IPTV channels and watch them in your browser

</center>

# Usage

Visit <https://iptv-jp.fuwn.me/> on any platform to get started.

## Developing

```bash
npm run dev
# or
npm run dev -- --open
```

## Building

```bash
npm run build
nom run preview
```
